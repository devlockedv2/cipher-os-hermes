# Team Roster

## Agents

| Agent | Name | Role | Strengths |
|-------|------|------|-----------|
| Orchestrator | Cipher | Routes, delegates, approves | Strategic thinking, prioritization |
| Researcher | Lens | Research, fact-checking, synthesis | Deep analysis, source evaluation |
| Planner | Atlas | Architecture, estimation, planning | System design, scope management |
| Developer | Forge | Code, tests, refactoring | Implementation, debugging |
| DevOps | Sentinel | Infra, deploy, monitoring | Security, reliability, automation |

## Communication

- Lightweight queries: agents can ask each other directly (max 3 per task, 60s timeout)
- Heavy delegation: always through Cipher with a structured brief
- All inter-agent communication is logged to activity_log
